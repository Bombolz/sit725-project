Javascript

Javascript is a programming language that is used by web browsers to create a dynamic and interactive experience for users. Javascript is versatile and can be used on both front-end and back-end development and is considered to be one of the most popular programming languages today. While the primary use of javascript is used to build websites, it can also be used to create browser based games and mobile applications if used alongside certain frameworks.




Html/Css

Html and css are front-end programming languages that are used to design the structure, layout, visual depiction and of a website. Html and css is essentially the face of a website, and creates a strong first impression on users visiting the site. A poorly designed website will quickly deter users while a professional one will earn their trust.




C#

C# is a popular general-purpose programming language comprising of front-end and back-end used for developing applications on the microsoft platform and requires .NET framework to work. It is particularly good for windows desktop applications and games, but can also be used for web development as well as mobile platforms.




Node.js

Node.js is a back-end javascript runtime programming language that is designed to build scalable network applications. It is particularly good at asynchronous event scenarios such as proxying different services with different response times or collecting data from multiple source points. It is a server side proxy where it can handle large amounts of simultaneous connections in a non blocking manner.
